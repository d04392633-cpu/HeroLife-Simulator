# HeroBattle Simulator (Go)

Простой консольный симулятор боя между героями, написанный на языке Go.
Проект создан в учебных целях для демонстрации разных подходов к работе с функциями, указателями и методами.

---

## Задание

В рамках задания требовалось реализовать **одну и ту же программу тремя разными способами**:

* без указателей
* с использованием указателей
* с использованием методов

Каждый вариант оформлен как отдельный проект.

---

## Реализации

### 1. `HeroBattle_17_Value`

* Без указателей
* Структуры передаются **по значению (копируются)**
* Функции возвращают результат

Пример:

```go
hero = attack(hero)
```

---

### 2. `HeroBattle_20_Pointer`

* Используются указатели (`*Hero`)
* Изменения происходят напрямую
* Функции ничего не возвращают

Пример:

```go
attack(&hero)
```

---

### 3. `HeroBattle_22_Methods`

* Используются методы структуры
* Указатели применяются внутри методов
* Более чистый и читаемый код

Пример:

```go
hero.attack()
```

---

## Логика игры

* Есть 2 героя (например: `Potato` и `Carrot`)
* У каждого есть:

  * имя
  * здоровье (health)

### Каждый ход:

1. Герои атакуют друг друга (случайный урон)
2. Затем восстанавливают здоровье
3. Игра продолжается до победы одного из героев

---

## Чему учит проект

* Работа со `struct` в Go
* Разница между передачей **по значению** и **по указателю**
* Использование указателей (`*` и `&`)
* Создание методов (`func (h *Hero) ...`)
* Базовая архитектура программы
